# Premium Quote Sticker Engine (Python Edition) 🐍

A high-fidelity Telegram quote sticker renderer re-engineered in pure Python.

### Features
- **Dynamic Grouping**: Bubbles adjust their corners based on message sequence.
- **Vector Beaks**: Authentic Telegram-style "needles" for messages.
- **Premium Emojis**: Automatic downloading and rendering of premium status emojis.
- **Rich Text Support**: Bold, Italic, Code, Underline, Spoiler, and Blockquotes.
- **UHD Rendering**: High-resolution output for crisp stickers.

### Configuration
Make sure to add your **`BOT_TOKEN`** in the **Space Secrets** (Settings > Variables and Secrets) to enable premium emoji features.
